# Synth Cookbooks Public

Public-safe Synth cookbook recipes and examples.

Cookbooks should only land here after review in `synth-cookbooks-private` confirms the recipe is reproducible without private credentials, internal datasets, unreleased behavior, or unredacted logs/screenshots.

## Layout

- `cookbooks/` - published recipes.
- `containers/` - public-safe benchmark harnesses, service wrappers, and task
  adapters used by published recipes.
- `assets/` - public-safe screenshots, diagrams, and generated media.
